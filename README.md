# Tripleten web_project_homeland

Galería TripleTen: De patria a patria
Este proyecto es una página web estática creada para el curso de TripleTen, que presenta la diversidad cultural de los profesionales de la comunidad a través de historias y fotografías de sus lugares de origen.

Enlace al Proyecto en GitHub Pages: https://jessy291099-gif.github.io/web_project_homeland/

Descripción
La página "De patria a patria" es un homenaje a los colegas de TripleTen que provienen de diversos rincones del mundo, desde Kentucky hasta Burundi, pasando por Gales y Ucrania. Incluye:

Un diseño limpio y moderno.
Una sección narrativa sobre la diversidad cultural.
Una cita inspiradora de Joseph Beuys.
Una invitación a la comunidad para compartir sus propias historias.
Tecnologías Utilizadas
HTML5: Estructura semántica.
CSS3: Estilos y diseño responsive.
